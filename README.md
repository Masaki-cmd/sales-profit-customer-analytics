# sales-profit-customer-analytics
# 売上・利益向上に向けたデータ分析とマーケティング戦略提案

## 概要

この分析は、`monthly_revenue.csv`データを用いて、売上と利益の向上を目的としたデータドリブンなマーケティング戦略を提案するものです。データの初期分析から始まり、特定の四半期の利益低迷要因の深掘り、顧客セグメント分析、そして最終的には顧客獲得コスト（CPA）を考慮した利益分析を通じて、新規顧客獲得とリピート顧客維持の両面から最適な投資配分と具体的な施策を導き出しました。

## 主要な分析と発見

### 1. データ初期分析と可視化

*   `monthly_revenue.csv`を読み込み、`japanize-matplotlib`を用いて日本語表示に対応させました。
*   売上、注文数、平均注文額、割引率、返品率、ユニーク顧客数、新規顧客数などの主要指標の月次推移を時系列で可視化しました。
*   初期の傾向として、売上成長の鈍化や利益率への影響（割引率の変動）が示唆されました。

### 2. 特定四半期（Q2, Q3）の利益分析

*   `推定利益 = 売上 × (1 - 平均割引率/100) × (1 - 返品率/100)`の式で推定利益を計算し、四半期ごとに可視化しました。
*   **Q2とQ3の推定利益が他の四半期と比較して低い**ことが判明しました。

### 3. Q2, Q3の利益低迷要因の深掘り

*   **割引率と返品率の分析**: Q2とQ3の割引率・返品率が他の四半期より特別に高いわけではないことを確認しました（特に返品率は全期間で0%）。これにより、利益低迷の主要因はこれらではないと結論付けられました。
*   **月次売上の詳細分析**: Q2（4-6月）とQ3（7-9月）の月次売上推移を分析した結果、**6月、7月、8月にかけて売上が低迷する季節的な傾向**が見られました。
*   **注文数とユニーク顧客数の分析**: 6月、7月、8月の注文数とユニーク顧客数を分析した結果、これらの月に**注文数とユニーク顧客数の両方が低迷**していることが確認されました。このことから、売上低迷の主な要因は販売量そのものの減少にあると特定されました。

### 4. 顧客セグメント分析 (新規・リピート)

*   新規顧客数とリピート顧客数（`unique_customers - new_customers`）の月次推移を可視化しました。
*   これらのグラフから、売上低迷期にどちらの顧客タイプに問題があるかを特定する手がかりを得ました。

### 5. 回帰分析による売上貢献度評価

*   **単回帰分析**: 売上を目的変数、リピート顧客数を説明変数とする分析で、リピート顧客1人あたりの売上貢献度（約147.6ドル）が統計的に有意であることを示しました（R-squared: 0.311）。
*   **重回帰分析**: 新規顧客数も加えた重回帰分析を実施し、モデルの決定係数（R-squared）が0.515に向上しました。
    *   `repeat_customers`の係数は約138ドル（p<0.001）
    *   `new_customers`の係数は約166ドル（p<0.001）
    *   新規顧客1人あたりの売上貢献度がリピート顧客よりもわずかに高いことが示唆されました。

### 6. CPA（顧客獲得コスト）を考慮した利益分析

*   **仮定CPA**: 新規顧客獲得CPAを50ドル、リピート顧客獲得CPAを10ドルと仮定しました。
*   **CPA考慮後利益の計算**: `推定利益 - 新規顧客CPA総額 - リピート顧客CPA総額`でCPA考慮後利益を計算し、月次推移を可視化しました。
*   **発見**: CPAを考慮すると推定利益が大幅に減少することが明らかになり、特に新規顧客獲得コストの総額が利益に大きな影響を与えることが示されました。

## 利益最大化戦略の提案

上記の分析結果に基づき、以下の戦略を提案します。

### 1. 新規顧客獲得とリピート顧客維持への投資配分の評価

*   **新規顧客**: 売上への貢献度は高いが、獲得コストも高い。CPA効率を重視し、ROIを最大化する必要があります。
*   **リピート顧客**: 売上への貢献度も高く、獲得コストは低い。安定した収益基盤として投資を強化すべきです。

### 2. 最適な投資配分に関する提案

*   **新規顧客獲得への投資**: CPA効率の高いチャネルやキャンペーンに重点を置き、ROIを最大化します。特に6月、7月、8月の低迷期には、集中的な新規獲得キャンペーンを計画します。
*   **リピート顧客維持への投資**: 低いCPAで安定した売上貢献が見込めるため、投資を強化し、LTV（顧客生涯価値）を最大化する施策を実施します。

### 3. 各顧客セグメントに特化したマーケティング戦略

#### 新規顧客獲得戦略

1.  **CPA最適化**: 各広告チャネルのCPAを定期的にモニタリングし、効率の悪いチャネルからは撤退または改善を図ります。
2.  **季節性への対応**: 6月、7月、8月に合わせた「サマーキャンペーン」など、ターゲット層の関心を引くプロモーションを強化します。
3.  **オンボーディングプログラムの強化**: 新規顧客がリピート顧客に転換する率を高めるための施策を実施します。

#### リピート顧客維持戦略

1.  **ロイヤルティプログラムの強化**: ポイントプログラムや会員ランクに応じた特典提供により、継続的な購買意欲を高めます。
2.  **パーソナライズされたコミュニケーション**: 購買履歴に基づいた商品レコメンデーションやコンテンツ配信でエンゲージメントを高めます。
3.  **顧客サポートの向上**: 迅速かつ質の高いカスタマーサポートで顧客満足度を向上させます。
4.  **アップセル・クロスセル戦略**: 既存顧客の購買傾向を分析し、より高単価な商品や関連商品を提案することで、平均注文額（AOV）とLTVを向上させます。

### 4. 提案戦略が「CPA考慮後利益」の向上に貢献するメカニズム

*   **新規顧客獲得戦略の最適化**: CPA効率の向上とオンボーディング強化により、無駄な広告費を削減し、長期的なLTVを向上させます。
*   **リピート顧客維持戦略の強化**: 既存顧客の購買頻度や購入単価を向上させることで、CPA考慮後利益を効率的に増加させ、収益基盤を強化します。

## 今後の課題とネクストステップ

*   **重回帰分析結果の深掘り**: 各係数のビジネス的意味の厳密な評価、多重共線性の可能性などをさらに詳細に検討します。
*   **CPAの妥当性検証**: 設定したCPAは仮定値であるため、実際のCPAデータを検証し、より現実的な数値で分析を行う必要があります。
*   **より詳細な顧客セグメント分析**: LTVやCPAに基づく顧客分類など、より詳細な顧客レベルのデータを用いたセグメント分析を検討します。
*   **戦略の効果測定と改善サイクル**: 提案された戦略が実際に利益にどのように貢献するかを測定し、PDCAサイクルを回すための具体的な指標や計画を策定します。
